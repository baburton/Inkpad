O que é o Inkpad?

O Inkpad é uma aplicação de ilustração vetorial, criada de raiz para o iPad. Suporta linhas, linhas compostas, texto, texto em linhas, imagens, grupos, máscaras, preenchimento com gradientes e camadas ilimitadas.

Inkpad é um software gratuito. Distribuído sob a Licença Pública Mozilla v2.0 que pode ser encontrada em http://mozilla.org/MPL/2.0. O código fonte pode ser encontrado em http://github.com/baburton/Inkpad.

Galeria

A galeria mostra uma miniatura de cada um dos seus desenhos. Toque numa miniatura para abrir, ver ou editar. Toque no nome por baixo da miniatura para a renomear.

Toque no botão + no canto superior direito do écran para criar um novo desenho.

Toque no botão Câmera para tirar uma fotografia e usá-la como fundo em novo desenho.

Importar

O Inkpadk pode importar imagens, fontes (TrueType e OpenType), SVG, SVGZ, e ficheiros do Inkpad através do Dropbox. Toque em Importar para aceder à conta Dropbox.

As fontes importadas aparecem no menu Fontes na Galeria, podem ser removidas caso não necessite mais delas. Na edição de um desenho, o menu Fontes mostra tanto as fontes importadas como as que vêm por defeito com a aplicação.

Toque no botão Amostras para reinstalar os desenhos de amostragem que vêm por defeito com o Inkpad.

Exportar

Toque em Selecionar para entrar no modo de seleção.

No modo de seleção, toque nas miniaturas para as selecionar (ou as deselecionar). Quando um ou mais desenhos estiverem selecionados, toque em Email para os enviar por email, ou toque em Dropbox para os enviar diretamente para a sua conta Dropbox.

Toque no botão Lixo para apagar os desenhos selecionados.

Toque no botão Feito para sair do modo de seleção.

Desenhar

Use o menu Editar para cortar, copiar, colar, apagar e alterar a seleção.

Use o menu Organizar para mover objetos para trás e para frente na pilha, agrupar e desagrupar objetos, distribuir objetos e alinhar objetos.

Use o menu Linhas para adicionar âncoras, remover âncoras, contornar por cima, juntar linhas, combinar linhas, fazer máscaras e colocar o texto em uma linha.

Use o menu de Ajustes de Cor para misturar cores de preenchimento nos objetos selecionados, ajustar a matiz, saturação, luminosidade e equilíbrio de cores do objeto selecionado. Também é possível desaturar e inverter as cores da seleção.

Quando misturar as cores, as formas em cada extremo da seleção devem conter cores de preenchimento (não gradientes). Por exemplo, quando as misturar horizontalmente, as formas mais à esquerda e mais à direita devem conter preenchimento de cor. Adicionalmente, pelo menos 3 formas devem ser selecionadas para libertar as opções de mistura.

Toque no botão Desfazer para desfazer uma alteração.

Toque no botão Refazer para refazer uma alteração.

No canto superior direito do écran tem mais opções para trabalhar nos seus desenhos:

Eles são respetivamente: Redimensionar para Caber, Importar Fotografia, Ajustes, e Partilhar/Exportar.

Ajustes

Use o painel Ajustes para ativar ou não a grelha, controlar o espaçamento da grelha, e indicar se os objetos se encaixam na grelha, se encaixam em pontos de linhas existentes, ou encaixam nas extremidades das linhas existentes. Também pode ativar ou desativar a régua, mudar as unidades de medida do seu desenho, e modificar as dimensões do seu desenho.

Quando Isolar a Camada Ativa está ativo, pode apenas editar a camada ativa. As outras camadas do seu desenho ficarão parcialmente escondidas. Isto torna mais fácil editar o conteúdo de uma camada sem acidentalmente danificar algo de outra camada.

Quando o Modo de Contorno estiver ativo, só consegue ver o esboço do objeto no desenho. Isto torna mais fácil selecionar objetos que estão escondidos sob outros objetos. Também reduz significantemente o tempo de renderização em desenhos complexos.

Visualização

Utilize o gesto padrão do iOS de aproximar ou afastar. Depois de começar o gesto para aproximar ou afastar pode levantar um dedo para mover o desenho sem alterar a visualização.

Se as unidades do desenho estão em pixéis, o Inkpad mostra o desenho no tamanho real quando o zoom estiver a 100% (como indicado na barra de navegação).

Se utiliza unidades de medida relacionadas com medidas físicas (polegadas, centímetros, etc.), o Inkpad mostra o desenho no tamanho de impressão quando o zoom estiver em 100%.

Ferramentas

O Inkpad tem um grande número de ferramentas para criar e modificar os seus desenhos.

A paleta de ferramentas flutua em cima da folha de desenho. Arraste-a para movê-la no écran. Caso esteja a atrapalhar num desenho mova-o para outro local.

Pode-se modificar o comportamento das diversas ferramentas, pressionando um segundo dedo no écran depois de começar a arrastar. Se aplicar um segundo toque depois de começar a arrastar irá aproximar a folha de desenho.

O comportamento de cada ferramenta é descrito nas seções seguintes.

Ferramentas de Seleção e Multi Seleção

A Ferramenta de Seleção seleciona um objeto quando toca nele. Qualquer seleção previamente feita será desselecionada.

A Ferramenta de Multi Seleção não anula as seleções anteriores, ela adiciona os novos objetos que tocar na seleção. Tocando num objeto selecionado a Ferramenta de Multi Seleção irá desselecioná-lo.

Selecione um grupo de objetos tocando numa parte vazia do desenho e arrastando à volta deles para abrangê-los. Um rectângulo escuro aparece para indicar a região selecionada.

Arraste os objetos selecionados para os mover na folha de desenho. Tocando com um segundo dedo depois de começar a arrastar constrange o movimento na horizontal, na vertical e na diagonal.

Quando uma linha estiver selecionada, pode editar as âncoras e controlar os pontos de direção. Como na ferramenta Caneta, descrita de seguida, pressionando um segundo dedo enquanto estiver controlando o ponto de direção faz com que o controle do ponto de direção oposto se mantenha fixo.

Toque duplo nas âncoras de uma linha para as converter entre vértices e curvas. Toque duplo no controle da alça para a remover.

Faça um toque duplo numa âncora e arraste-a para fazer que o controle da alça fique espelhado.

Caneta

A Caneta é uma ferramenta poderosa para criar linhas no Inkpad. Pode usá-la para fazer qualquer forma. Para começar, selecione a Caneta na Paleta de Ferramentas.

Toque uma vez na folha do desenho para adicionar um ponto (âncora) do primeiro vértice da sua forma. Uma série de toques irá produzir um polígono com cantos agudos.

Para fazer um canto curvo, toque e arraste. Irá ver um conjunto de controles dos pontos de direção aparecerem determinando a forma da curva. Quanto mais distantes ficarem os controles dos pontos de direção do ponto âncora, mais extrema será a curva.
Pode também mover os controles dos pontos de direção de forma indepentende. Assim que começar a arrastar, toque no écran com um segundo dedo para manter o controle do ponto de direção fixo. Isso permite criar dois segmentos curvos que formam um vértice.

Toque na primeira âncora dessa linha para a fechar.

Toque duplo na última âncora da linha para terminar a forma sem a fechar.

Para continuar a edição de uma linha não fechada, primeiro selecione-a com a ferramenta de Seleção. Depois escolha a ferramenta Caneta e toque num ponto da extremidade da linha. Pode agora continuar o desenho da linha ou tocar na outra extremidade para a fechar.

Ferramenta Pincel

A ferramenta Pincel desenha formas livremente na folha de desenho. Esta é uma boa maneira para esboçar e pintar.

Ferramenta Borracha

A ferramenta Borracha apaga partes das formas no desenho. Funciona em linhas não agrupadas ou compostas.

A ferramenta Borracha apenas funciona em formas selecionadas.

Pode ajustar a espessura da borracha através do painel que aparece na parte inferior do desenho, quando a ferramenta Borracha está selecionada.

Ferramenta Acrescentar Âncora

A ferramenta Acrescentar Âncora adiciona pontos âncora nas formas existentes no desenho.

Toque na borda de uma forma para acrescentar um novo ponto âncora.

A ferramenta Acrescentar Âncora seleciona a forma caso seja necessário, mas é mais fácil acrescentar pontos âncora se selecionar primeiro a linha, assim pode ver os pontos âncora existentes.

Ferramenta Tesoura (Ferramenta Dividir Linhas)

A Ferramenta Tesoura corta linhas já existentes.

Para dividir uma linha, toque na borda que deseja cortar. Para dividir num ponto âncora, toque nele.

Se a linha está fechada, ela abre e ambos os pontos das extremidades são localizados no local do toque. Se a linha não está fechada, ele divide-se em duas linhas no local do toque.

Ferramentas de Formas

As ferramentas de Formas criam retângulos, ovais, estrelas, polígonos, espirais e segmentos de reta.

Toque e mantenha pressionado na forma ativa para aceder às outras formas. Uma paleta secundária aparece para aceder às outras formas.

Por defeito, os retângulos e as ovais não estão ajustados a nenhuma relação de aparência. Se quiser um quadrado perfeito ou um círculo perfeito, toque no écran com um segundo dedo depois de arrastar para construir a forma.

É possível ajustar o raio dos vértices dos retângulos, o número de pontas de uma estrela, o número de curvas em uma espiral, ou o número de lados de um polígono através do painel que aparece na parte inferior do écran quando seleciona a ferramenta.

Se tocar no écran com um segundo dedo quando constrói uma estrela, o raio interior da mesma mantêm-se constante, permitindo ter uma estrela mais ou menos pontiaguda.

Se tocar no écran com um segundo dedo quando constrói um segmento de reta, a linha fica constrangida a ser desenhada horizontalmente, diagonalmente ou verticalmente.

Ferramenta Conta-Gotas

A ferramenta Conta-Gotas recolhe os atributos de estilo de objetos existentes no desenho. Ela recolhe e aplica qualquer propriedade que possa ser editada no painel de estilos descrito na secção seguinte.

Para utilizar esta ferramente, arraste o seu dedo por cima dos objetos do seu desenho de maneira a apanhar o estilo que quer aplicar.

Se nada é selecionado quando arrasta a ferramenta Conta-Gotas, o estilo será aplicado ao próximo objeto que criar. Se algum objeto estiver selecionado, o estilo será aplicado aos objetos selecionados.

Ferramenta Texto

A Ferramenta de Texto cria objetos de texto.

Toque e arraste para criar uma caixa de texto na folha de desenho. Quando acabar de arrastar, o teclado de écran aparece, permitindo a digitação de texto. Para acabar toque em qualquer sítio na folha de desenho.

Para editar o texto, selecione a ferramenta Texto, toque no texto, ou escolha as ferramentas de Seleção e faça duplo toque no objeto de texto.

Ferramentas de Escalar e Rodar

As ferramentas de Escalar e Rodar transformam os objetos selecionados.

Um simples toque utilizando estas ferramentas move o ponto de deslocação (inicialmente ele aparece no centro da seleção). Os objetos escalam-se ou rodam quando o arrastar.

A ferramenta Escalar mantêm a relação de aspeto dos objetos selecionado, mas tocando com um segundo dedo e mantendo enquanto arrasta provoca um escalonamento não uniforme.

A ferramenta Rodar, roda livremente, exceto se tocar com um segundo dedo enquanto arrasta, nesse caso a ferramenta funciona rodando em ângulos com incrementos de 45 graus.

Estilos de Objetos

Toque nos botões dos estilos de objetos para modificar a aparência dos objetos selecionados:

Eles são respetivamente: Fonte, Sombra e Opacidade, Traço, Preenchimento, e Amostras.

O botão Sombra e Opacidade indica a cor da sombra e a sua direção, assim como a opacidade do objeto. As cores indicadas nos botões Traço e Preenchimento são valores usados quando são criadas novas formas.

Amostras

O painel Amostras guarda cores e gradientes para posterior uso.

O controle Sombra/Linha/Preenchimento no fundo do painel determina o estilo que é guardado quando toca no botão + no lado superior direito do painel. Também determina qual é o estilo quando toca numa amostra do painel.

Como os gradientes apenas funcionam em preenchimentos de formas, tocando em uma amostra de um gradiente, ele preenche a forma apesar de outras especificações.

Gradientes

Um gradiente é um preenchimento de uma forma com transições entre duas ou mais cores. Um gradiente é definido por um conjunto de pontos de cor. Cada ponto de cor indica uma cor e a sua posição no gradiente.

Para aplicar um gradiente, abra o painel Preenchimento e toque no botão Gradiente na barra do título.

Toque num ponto de cor para selecioná-lo. Utilize os botões deslizantes de cor para alterar os pontos de cor. Arraste o ponto de cor para alterar a sua posição no gradiente.

Para inverter a direção do gradiente, deslize para a direita ou para a esquerda na barra do gradiente (a área por cima dos pontos de cor).

Para distribuir os diferentes pontos de cor num gradiente, faça duplo toque na barra do gradiente.

Para remover um ponto de cor de um gradiente, arraste-o para cima ou para baixo.

Para inserir um ponto de cor, toque no espaço em branco entre os pontos de cor (por baixo da barra do gradiente).

Camadas

O Inkpad suporta um número ilimitado de camadas. As camadas são úteis para organizar desenhos complexos. Cada componente do desenho pode ficar numa camada, fazendo com que a tarefa de edição desse elemento seja facilitada.

Pode apagar, duplicar, reorganizar, renomear, proteger e esconder as camadas. Pode também controlar a transparência de cada camada.

Toque no botão Camadas na barra para ver as camadas do desenho.

Juntar Linhas

Para unir duas linhas abertas, selecione-as e escolha Unir Linhas no menu Linha.

As linhas são fechadas juntando os dois extremos mais próximos, assim deve alinhá-los na maneira que pretende que eles se unam antes de escolher Unir Linhas.

Se o o ponto oposto terminal coincide após a união, a linha nova é também fechada.

Linhas Compostas

Uma linha composta é uma linha composta por múltiplas linhas. É útil para a criação de formas que contenham buracos.

Para criar uma linha composta, seleccione uma ou mais linhas e/ou linhas compostas e escolha Combinar Linhas no menu Linha.

Para transformar uma linha composta em linhas simples, escolha Separar Linhas no menu Linha.

A direção relativa de cada linha secundária numa linha composta determina quais as linhas secundárias que são preenchidas. Se a linha tem a mesma direção, são todas fechadas.

Se combinou duas linhas na esperança de criar um buraco, mas o buraco está preenchido, selecione uma das linhas na linha composta e escolha Reverter a Direção da linha no menu Linha.

Colocar Texto numa Linha

Para colocar texto numa linha não composta já desenhada, seleccione-a (ou uma linha e um objeto de texto existente) e escolha Colocar Texto numa Linha no menu Linha.

Um editor de texto aparece, permitindo editar o texto na linha. Pode reativar mais tarde este editor efetuando duplo toque na linha com uma das ferramentas de Seleção (ou tocar na Linha com a ferramenta de Texto).

A linha é editável como se tratasse de uma linha normal, mas as especificações do estilo só se aplicam ao texto.

Arraste a barra para o início do texto para ajustar o ponto de começo do texto na linha.

Para o texto aparecer no outro lado da linha, escolha Inverter o Sentido da Linha a partir do menu Linha.

Máscaras

Uma máscara é uma linha, uma linha composta, ou objeto de texto que corta o conteúdo de outros elementos desenhados.

Para criar uma máscara, selecione os objetos que quer mascarar e a linha ou texto que quer usar como máscara (o objeto que vai servir de máscara deve estar por cima dos objetos selecionados). Escolha Máscara no menu Linha.

Para desfazer a máscara dos objetos, selecione os objetos com máscara e escolha Remover Máscara no menuLinha.

Uma linha que serve de máscara pode ter um preenchimento e uma espessura como uma linha normal.
