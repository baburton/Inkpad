Qu'est ce qu'Inkpad ?

Inkpad est une application de dessin vectoriel conçu à partir de zéro pour l'iPad. Il prend en charge les tracés, les tracés composés, les textes, les textes sur des tracés, les images, les groupes, des masques, les dégradés, et un nombre illimité de calques.

Inkpad est un logiciel libre. Il est distribué sous la license "Mozilla Public License v2.0" qui se trouve à http://mozilla.org/MPL/2.0. Le code source se trouve sur http://github.com/baburton/Inkpad.

Galerie

La galerie affiche une vignette pour chacun de vos dessins. Appuyez sur une vignette pour ouvrir le dessin afin de le visualiser ou de le modifier. Appuyez sur le titre au-dessous de la vignette pour entrer dans un nouveau titre.

Appuyez sur le + dans le coin supérieur droit de l'écran pour créer un nouveau dessin.

Appuyez sur le boutton l'appareil photo pour prendre une photo et l'utiliser comme arrière-plan pour un nouveau dessin.

L'import

Inkpad peut importer des images, des polices de caractère (TrueType et OpenType), SVG, SVGZ, et des fichiers Inkpad depuis Dropbox. Appuyez sur Importer pour accéder à votre Dropbox.

Les polices de caractère importés apparaissent dans la bibliothèque des Polices de caractères du panneau où ils peuvent être supprimés si vous ne souhaitez plus les utiliser. Lors de l'édition d'un dessin, le panneau des polices de caractère affiche les polices importées mélangées avec les polices intégrées.

Appuyez sur les exemples pour réinstaller les œuvres inclus avec Inkpad.

L'export

Appuyez sur Sélectionner pour passer en mode de sélection.

En mode de sélection, appuyez sur les vignettes pour les sélectionner (ou désélectionner). Quand un ou plusieurs dessins sont sélectionnés, appuyez sur Adresse Électronique pour les envoyer par email, ou tapez sur Dropbox pour les envoyer directement sur votre compte Dropbox.

Appuyez sur le boutton de la Corbeille pour supprimer les dessins sélectionnés.

Appuyez sur le boutton OK pour quitter le mode de sélection.

Dessin

Utilisez le bouton du menu Editer pour couper, copier, coller, supprimer, et modifier la sélection.

Utilisez le bouton du menu Arranger pour déplacer les objets vers l'arrière et vers l'avant de la pile, grouper et dissocier les objets, distribuer les objets, et aligner.

Utilisez le bouton du menu Tracer pour ajouter/supprimer des points d'ancrages, ajouter des contours, combiné des tracés, créer des masques, et placer du texte sur un tracé.

Utilisez le bouton du menu Ajustement de la couleur pour mélanger les couleurs de remplissage au travers des formes sélectionnées, régler la teinte, la saturation, la luminosité et la balance des couleurs des objets sélectionnés, et désaturer et inverser les couleurs de la sélection.

Lorsque vous utilisez le mélange des couleurs, les formes à chaque extrémité de la sélection doivent avoir des remplissages de couleur unis (pas de dégradés). Par exemple, lors de la fusion horizontale, les formes les plus à gauche et les plus à droite doivent avoir des remplissages de couleur unis. De plus, au moins trois formes doivent être sélectionnés de façon à activer les options de fusion.

Appuyez sur le bouton Annuler pour annuler une modification.

Appuyez sur le bouton Rétablir pour rétablir une modification.

En haut à droite de l'écran, il y a plus d'options pour travailler avec votre dessin :

Respectivement, il y a : l'option Ajuster, Placer une Photo, Paramètres, et Partager/Exporter.

Paramètres

Utilisez le panneau Paramètres pour afficher/cacher la grille, contrôler l'espacement de la grille, indiquer si vous souhaitez que les objets dessinés soient aligner sur la grille, aligner sur les points des différents tracés existants, ou encore de les alginer sur les bords des tracés existants. Vous pouvez également afficher et cacher les règles, changer les unités de mesure de votre dessin, et de modifier les dimensions de celui-ci.

Lorsque Isoler le Calque Actif est activé, vous ne pouvez éditer que le calque actif. Les autres calques dans votre dessin seront partiellement masqués. Cela permet de simplifier rla modification du contenu d'un calque sans toucher accidentellement quelque chose dans un autre calque.

Lorsque le Mode Contour est activé, vous ne pouvez voir que les contours des objets présent dans votre dessin. Cela permet de simplifier la sélection des objects qui sont masqués par d'autres objets. Elle réduit également de manière significative le temps de rendu pour les dessins complexes.

Zoom

Utilisez geste standard d'iOS du pincement pour zoomer et dézoomer. Une fois que vous avez commencé un geste de pincement, vous pouvez lever un doigt pour déplacer le dessin autour sans zoom.

Si les unités de mesures de votre dessin sont en pixels, Inkpad affiche le dessin à la taille réelle des pixels lorsque le niveau de zoom est de 100% (comme indiqué dans la barre de navigation).

Si vous utilisez des unités qui ne correspondent pas à une mesure physique (pouces, centimètres, etc), Inkpad affiche le dessin au format d'impression lorsque le niveau de zoom est de 100%.

Outils

Inkpad fournit un certain nombre d'outils pour créer et modifier votre dessin.

La palette d'outils flotte toujours au-dessus de la toile. Faites-la glisser pour la déplacer autour de l'écran. Donner un petit coup rapide pour l'enlever de votre chemin si nécessaire.

Vous pouvez modifier le comportement de nombreux outils en maintenant un deuxième doigt sur ​​le dessin après avoir commencer le faire glisser. Si vous appliquez le deuxième contact avant de commencer à glisser, vous zoomerez sur sur le dessin.

Le comportement de chacun des outils est décrit dans les sections ci-dessous.

Outils de Selection et de Multi-Selection

L'outil de sélection sélectionne toujours un seul objet lorsque vous appuyez sur le dessin. Tous les objets précédement séléctionnés sont déselectionnés.

L'outil de multi-sélection ne rejette pas la sélection précédente, mais ajoute de nouveaux objets à la sélection. Le fait de toucher un objet sélectionné avec l'outil multi-sélection le désélectionne.

Sélectionnez un groupe d'objets en appuyant sur une partie vide du dessin et faire glisser pour les inclure. Une boîte (appelé un chapiteau) apparaît pour indiquer la zone de sélection.

Faites glisser les objets sélectionnés pour les déplacer autour de votre dessin. Vous pouvez contraindre le mouvement horizontalement, en diagonale ou verticalement en maintenant un deuxième doigt sur ​​la toile après avoir commencé le glissement.

Quand un seul tracé est sélectionné, vous pouvez modifier les points d'ancrages et les points de contrôle de celui-ci. Comme avec l'outil Plume décrit après, en maintenant un deuxième doigt lorsque vous faites glisser un point de commande force le point de commande opposée à rester fixe.

Tapez deux fois sur les points d'ancrages d'un tracé pour les convertir entre les coins et les courbes. Tapez deux fois sur un point de commande pour le supprimer.

Tapez deux fois et faites glisser un point d'ancrage pour modifier les points de contrôle en miroir les uns des autres.

Outil Plume

L'outil Plume est le moyen le plus puissant pour créer des tracés dans Inkpad. Vous pouvez l'utiliser pour définir toute les formes possibles. Pour commencer, sélectionnez la Plume de la palette d'outils.

Appuyez une fois sur le dessin pour ajouter un point d'angle unique (une ancre) à votre forme. Une série d'appui simple produira un polygone à angles vifs.

Pour faire un bord incurvé, appuyez et faites glisser. Vous verrez une série de poignées de contrôle apparaitrent qui déterminent la forme de la courbe. Les poignées de commande les plus éloignées sont issus du point d'ancrage, la plus extrême celle de la courbe.

Vous pouvez également déplacer les poignées de contrôle indépendamment les unes des autres. Une fois que vous commencez à faire glisser, toucher le dessin avec un deuxième doigt pour maintenir la poignée de contrôle opposée fixe. Cela vous permet de créer deux segments courbés qui forment un coin.

Appuyez sur le premier point d'ancrage dans le tracé pour le fermer.

Appuyez deux fois sur le dernier point ancrage du tracé afin de finir la forme sans la fermer.

Pour continuer la modification d'un chemin non fermé existant, vous devez le sélectionner avec l'outil de Sélection. Ensuite, choisissez l'outil Plume et appuyez sur l'un des points du tracé. Vous pouvez maintenant étendre le tracé ou appuyer sur le point d'extrémité opposée pour le fermer.

Outil Main Levée

L'outil Main Levée permet de dessiner des formes libre sur le calque. C'est un excellent outils pour griffonner et peindre.

Outil Gomme

L'outil Gomme permet d'effacer des parties de formes dans votre dessin. Il fonctionne sur n'importe quel tracé dissocié ou composé.

Si il y a une sélection, l'outil Gomme ne fonctionnera que sur les objets sélectionnés.

Vous pouvez ajuster la largeur de la gomme en passant par le petit panneau de configuration qui apparaît au bas de la toile lorsque l'outil Gomme est sélectionné.

Outil Ajouter des Points d'Ancrage

L'outil Ajouter des Points d'Ancrage ajoute des points d'ancrage à des formes existantes de votre dessin.

Appuyez sur le bord d'une forme pour ajouter un nouveau point d'ancrage.

L'outil Ajouter des Points d'Ancrage sélectionne la forme si cela est nécessaire, mais il est plus facile d'ajouter des ancres si vous sélectionnez d'abord le tracé afin que vous puissiez voir les points d'ancrage déjà existants.

Outil Ciseaux

L'outil Ciseaux permet de couper les tracé existants.

Pour couper un tracé, il suffit d'appuyer sur le bord que vous souhaitez couper. Pour couper au niveau d'un point d'ancrage, il faut appuyez dessus.

Si le tracé a déjà été fermé, le tracé s'ouvre automatiquement et les deux points d'extrémités seront situés à l'emplacement du point de toucher. Si le tracé n'a pas été fermé, il sera divisé en deux tracé distinct au niveau de la touche.

Outil Forme

L'outil Forme permet de créer des rectangles, des ovales, des étoiles, des polygones, des spirales et des lignes.

Appuyez et maintenez l'outil Forme actif dans la palette d'outils pour y accéder. Une sous-palette apparaît qui vous permet de choisir une forme différente.

Par défaut, les rectangles et les ovales ne sont pas limités à ratio particulier. Si vous voulez un carré ou un cercle parfait, touchez l'écran avec un deuxième doigt après que vous ayez commencé à faire glisser la forme.

Vous pouvez régler le rayon des coins du rectangle, le nombre de points dans les étoiles, le taux de décroissance des spirales, et le nombre de côtés dans les polygones à l'aide un petit panneau de configuration qui apparaît au bas de l'interface lorsque vous sélectionnez l'outil correspondant.

Si vous touchez l'écran avec un deuxième doigt tout en faisant glisser une étoile, le rayon intérieur de celle-ci est maintenue constante, ce qui vous permet de faire en sorte que l'étoile sera plus ou moins pointu.

Si vous touchez l'écran avec un deuxième doigt tout en faisant glisser une ligne, la ligne sera limitée dans le sens horizontale, diagonale ou verticale.

Outil Pipette

L'outil Pipette collecte les attributs de style des objets existants dans votre dessin. Il prend et applique les propriétés qui peuvent être modifiées dans les panneaux de style décrites dans la section suivante.

Pour utiliser la pipette, faites glisser votre doigt sur les objets de votre dessin jusqu'à ce que vous avez ramassé le style que vous souhaitez appliquer.

Si rien n'est sélectionné lorsque vous faites glisser la pipette, les styles seront appliqués à l'objet suivant que vous créez. Si quelque chose est sélectionnée, les modèles sont appliqués à aux objets sélectionnés.

Outil Texte

L'outil Texte permet de créer des objets contenant du texte.

Tapez et faites glisser pour créer une zone de texte sur la toile. Lorsque vous avez terminé le déplacement, le clavier virtuel apparaît, vous permettant de modifier le texte. Pour arrêter l'édition, appuyez n'importe où sur la toile.

Pour modifier à nouveau le texte, sélectionnez l'outil Texte, puis tapez sur le texte voulu, ou choisissez l'un des outils de sélection et appuyez deux fois sur l'objet texte.

Outil Redimensionnement et Rotation

L'outil Redimensionnement et Rotation permet de transformer les objets sélectionnés.

Une simple pression en utilisant l'un de ces outils permet de déplacer le point de pivot (initialement il apparaît au centre de la sélection). Les objets se redimensionnent ou tournent le long du pivot lorsque vous faites les faites glisser.

L'outil Redimensionnement maintient le ratio d'aspect des objets sélectionnés, cependant en maintenant un deuxième doigt tout en faisant glisser l'autre doigt permet d'effectuer un redimensionnement non uniforme.

L'outil Rotation permet de tourner librement, sauf si vous maintenez un deuxième doigt tout en faisant glisser, dans ce cas, l'outil limite la rotation à 45°.

Styliser les Objets

Appuyez sur les boutons de style dans la barre d'outils pour modifier l'apparence des objets sélectionnés :

Ils sont, respectivement dans : Police de caractère, Ombre et Opacité, Contour, Remplissage, et Nuanciers.

L'Ombre et Opacité indique la couleur actuelle de l'ombre et de sa direction, ainsi que l'opacité de l'objet. Les couleurs indiquées dans les boutons Contour et Remplissage sont les valeurs utilisées lors de la création de nouvelles formes.

Nuanciers

Le Nuancier enregistre couleurs et des dégradés pour une réutilisation ultérieure.

Les commandes Ombre et Opacité / Contour / Remplissage au bas de l'écran détermine quel style est enregistré lorsque vous appuyez sur le bouton + dans le coin supérieur droit du panneau. Il détermine également quel style est utilisé lorsque vous appuyez sur l'un des échantillons dans le panneau.

Comme les gradients fonctionnent comme du remplissage, en tapant sur une nuance de dégradé, cela applique le style de remplissage indépendamment du réglage.

Dégradés

Un dégradé est un remplissage qui assure la transition entre deux couleurs ou plus. Un dégradé est définie par un jeu d'étapes de couleurs. Chaque étape spécifie une couleur et une position dans le gradient.

Pour appliquer un dégradé, ouvrez le panneau Remplissage et appuyez sur l'onglet Dégradé la barre de titre.

Appuyez sur une étape de couleur pour la sélectionner. Utilisez les curseurs de couleur en haut de l'écran pour modifier la couleur de l'étape. Faites glisser l'étape de couleur pour changer sa position dans le gradient.

Pour inverser la direction du gradient, balayez vers la gauche ou vers la droite dans la barre de dégradé (la zone au-dessus des étapes de couleurs).

Pour répartir les étapes dans un gradient, tapez deux fois sur la barre de dégradé.

Pour ajouter une nouvelle étape de couleur, appuyez sur l'espace vide entre les étapes de couleur (au-dessous de la barre de dégradé).

Pour supprimer une étape de couleur du gradient, faites glisser vers le haut ou vers le bas.

Calques

Inkpad supporte un nombre illimité de couches. Les calques sont utiles pour organiser des dessins complexes. Chaque composant du dessin peut être sur son propre calque, ce qui le rend beaucoup plus facile à modifier.

Vous pouvez supprimer, dupliquer, réorganiser, renommer, verrouiller et masquer les calques. Vous pouvez également contrôler la transparence de chacune d'entre elle.

Appuyez sur les bouton des Calques dans la barre d'outils pour voir les calques dans votre dessin.

Joindre les Tracés

Pour joindre deux tracés non fermés, sélectionnez les deux tracés et choisir Joindre les Tracés dans le menu Tracer.

Les tracés sont rejoints en utilisant leurs plus proches points de terminaison, de sorte que vous pouvez choisir l'alignement avant de Joindre les Tracés.

Si les extrémités opposées du nouveau tracé coïncident après assemblage, le nouveau tracé est fermé.

Grouper les Tracés

Un groupement de tracé est un tracé composé de plusieurs sous-tracés. Ils sont utiles pour créer des formes qui contiennent des trous.

Pour créer un tracé composé, sélectionnez deux ou plusieurs tracés et/ou des tracés composés et choisissez Grouper les Tracés dans le menu Tracer.

Pour dégroupper un tracé composé en tracé simples, choisissez Séparer les Tracés dans le menu Tracer.

L'orientation relative de chaque sous-tracé dans un tracé groupé détermine les sous-tracé qui sont remplis. Si tous les tracés s'étendent dans la même direction, ils sont tous remplis.

Si vous avez combiné deux tracés dans l'espoir de faire un trou, mais que le trou est rempli, sélectionnez l'un des deux tracé dans le tracé groupé et choisissez Inverser la Direction du Tracé dans le menu Tracer.

Placer du Texte sur un Tracé

Pour placer du texte sur un tracé, sélectionnez un tracé non-groupé (ou un tracé et un objet texte existant) et choisissez Placer du Texte sur le Tracé dans le menu Tracer.

Un éditeur de texte apparaît, vous permettant de modifier le texte sur le tracé. Réactiver cet éditeur plus tard en faisant une double touche sur le tracé avec l'un des outils de sélection (ou en tapant sur le tracé avec l'outil texte).

Le tracé est modifiable comme tout tracé normal, mais les paramètres de style s'appliquent uniquement au texte.

Faites glisser la barre de démarrage qui apparaît au début du texte pour ajuster la position de départ du texte sur le chemin.

Pour faire apparaître le texte de l'autre côté du chemin, choisissez Inverser la Direction du Tracé dans le menu Tracer.

Les Masques

Un masque est un tracé, composé, ou un objet texte qui attache le contenu des autres éléments de dessin.

Pour créer un masque, sélectionnez les objets que vous souhaitez masquer et le tracé ou l'objet texte que vous souhaitez utiliser comme masque (l'objet de masquage doit être l'objet le plus haut sélectionné). Choisissez Masquer dans le menu Tracer.

Pour démasquer des objets, sélectionnez l'objet masqué et choisissez Afficher dans le menu Tracer.

Un masque de tracé peut avoir un fond et un contour, comme tout chemin normal.
